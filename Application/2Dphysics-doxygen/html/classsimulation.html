Detailed Description

template<unsigned int N>
class simulation< N >

Simulation ADT.

This is the simulation class, it simulates one or multiple particles on a window.
It calculates an acceleration (vector), which is used to update the speed of the particles.
The class can contain a n number of particles. An increase in particles can harm the peformance.
The needed functions and constructor are provided.

Constructor simulation.

This constructor creates a simulation object.
You are obligated to give w (window&), mpu (MPU6050&), particles (array).
You can change the max_accel, accel_threshold and max_speed but this not necessary.
A change in max_accel or max_speed will change the behaviour of the particles in the simulation.
A change in threshold will make the simulation more or less sensitive.
Call example: auto simu = simulation(window, mpu, arr_particles, 50, 6);

Update acceleration.

This function updates the acceleration of the simulation.
If the angle given by the MPU6050 is above the threshold, than it calculates the acceleration.
The acceleration is never greater than max_acceleration.
Call example: simu.updateAcceleration();
