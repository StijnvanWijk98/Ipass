Detailed Description

Vector ADT.

This is an ADT that implements a vector based on floats.
The vector contains two floats.
It contains the needed operators to be used in the application.

◆ magnitude()

float vector::magnitude ( )

output operator vector to std::ostream

This operator streams the vector to a std::ostream.
The operator will give the following result in the ostream: (x,y)
The x and y values are printed in a 7 decimal precision. Magnitude vector

This function returns the magnitude of the vector.
In other words, it returns the length of the vector.
The magnitude is the sqrt(sq(x)+sq(y)). Call example: float magn = vector.magnitude();
