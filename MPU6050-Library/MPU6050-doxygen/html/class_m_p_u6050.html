Detailed Description

MPU6050 interface library.

This is a library for the MPU6050.
This library implements a accelerometer_interface.
The library also interfaces the gyroscope en temperature feature of the MPU6050.
The communication protocol is I2C.
The appropriate constructor and functions are given.

Constructor MPU6050.

This constructor creates a MPU6050 object.
You are obligated to give a i2c_bus.
When the AD0 is connected to VCC, the address should be 0x69.
Call example: auto mpu = MPU6050(bus_bit_banged);

void MPU6050::calibrate ( unsigned int  sample_rate = 200)

Calibrate MPU6050.

This function calculates the error values of the different sensors.
It is optional to call this function.
It is done by taking multiple measurements, and taking the avarage of those measurements.
For the calbration to work properly, you must be sure that the MPU6050 is level.
Call example: mpu.calibrate(400);

)

Set range accelerometer and gyroscope.

This function makes you able to set the range of the accelerometer and gyroscope.
The range is to amount of change the sensor can measure.
If range is bigger, the precision is less.
It expects acc_full_range and a gyro_full_range.
Call example: mpu.setFullRange(acc_full_range::range_8g, gyro_full_range::range_1000);

◆ whoAmI()

void MPU6050::whoAmI ( )

Print WhoAmI register.

This function prints the decimal value that is in the WhoAmI register.
The decimal value should be the same as the address.
This function can be used to check if the MPU6050 is working correctly.
Call example: mpu.whoAmI(); (mpu is the mpu6050 object)
